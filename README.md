# ShadowTrace
A Minecraft mod for detecting X-ray mining by visualizing mined blocks.

## Features
- Visual block outlines for mined blocks.
- Color-coded blocks based on type (e.g., diamonds, emeralds).
- Integration with CoreProtect for logging and detection.

## Dependencies
ShadowTrace uses the following third-party libraries:

- **[MaLiLib](https://github.com/sakura-ryoko/malilib)** (LGPL-3.0): Used for optimized rendering of block outlines.
  MaLiLib is dynamically linked, and its source code is available [here](https://github.com/sakura-ryoko/malilib).
